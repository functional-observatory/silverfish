# Silverfish

> Graphql Service to manage bookmarks

## Prerequisites

- [jdk](https://adoptopenjdk.net/?variant=openjdk11&jvmVariant=openj9)
- [sbt](https://www.scala-sbt.org/download.html)

## Usage

- Run server

```shell
$ sbt run
```
